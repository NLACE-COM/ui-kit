# Catálogo de imágenes AI — `assets/imagery/`

130 imágenes generadas con IA (Midjourney) para secciones tecnológicas y de producto de NLACE.

**Base URL:** `https://raw.githubusercontent.com/NLACE-COM/ui-kit/main/assets/imagery/`
**Archivos:** `ai-01.png` → `ai-130.png` (dos dígitos hasta `ai-99.png`, tres desde `ai-100.png`)
**Formato:** mayoría 16:9 horizontal; `ai-35`–`ai-37` son cuadradas.

La guía de estilo y la plantilla para generar imágenes nuevas del set están en
[`DESIGN.md` § Imágenes AI](../../DESIGN.md).

## Grupos temáticos

Cada grupo corresponde a un prompt de origen distinto (las variantes de un mismo prompt
comparten composición y tema). Útil para elegir rápido por concepto:

| Tema / concepto del prompt original | Imágenes |
|---|---|
| Empresas y el uso de la IA (genérico — el corazón del set) | `ai-01`–`ai-04`, `ai-15`–`ai-17`, `ai-19`, `ai-21`–`ai-28`, `ai-30`, `ai-84`–`ai-122` |
| Hombre entre dos caminos (decisión, encrucijada) | `ai-05`–`ai-07`, `ai-48` |
| Revolución humana frente a la digitalización | `ai-08` |
| Sin prompt registrado (mismo lenguaje visual) | `ai-09`–`ai-11`, `ai-18`, `ai-20`, `ai-29`, `ai-31`–`ai-34` |
| Empresas chilenas y el uso de la IA | `ai-12`–`ai-14`, `ai-83` |
| Figura humana diminuta ante arquitectura abstracta (3D render; cuadradas) | `ai-35`–`ai-38` |
| Red social vs. lugar vacío (contraste, dualidad) | `ai-39`–`ai-45` |
| Agentes de IA trabajando | `ai-46`–`ai-47` |
| Personas enfrentándose a la IA (cabezas monumentales) | `ai-49`–`ai-52` |
| La IA trabajando para las personas | `ai-53`–`ai-58` |
| Personas siendo reemplazadas (filas, bustos seriados) | `ai-59`–`ai-62` |
| La IA como aliada (encuentros humano-máquina) | `ai-63`–`ai-69` |
| Empresas agobiadas por los procesos | `ai-70`–`ai-79` |
| Falta de productividad empresarial | `ai-80`–`ai-82` |
