Campo de texto con label, estados de error/éxito y texto de ayuda.

`<Input label="Email" placeholder="tu@empresa.com" hint="Usaremos esto para contactarte" />`

- `label`, `hint`, `error` (string: pinta borde rojo y muestra el mensaje), `success` (boolean: borde verde).
- Pasa cualquier atributo nativo de `<input>` (`type`, `value`, `onChange`, `disabled`, …).
