# NLACE — instrucciones del proyecto

## Reglas de marca
- **Nunca usar fondos negros / casi-negros (#0f1011, #161719, etc.) como fondo de superficies, bloques o footers.** Preferir fondos claros: blanco `#ffffff`, off-white `#efefef`, superficie `#f4f4f0`. El negro `#0f1011` solo es válido como color de texto/tinta o como swatch de documentación de la paleta — no como fondo de relleno.

## Propuestas
- **Las propuestas no deben tener portada.** Si excepcionalmente llevan una apertura, NUNCA usar fondo negro y NUNCA poner el logo centrado (logo siempre arriba a la izquierda).
